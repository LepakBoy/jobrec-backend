# Project Backgorund

JobRec :
JobRec Adalah salah satu aplikasi lamaran pekerjaan secara online yang bertujuan untuk mempermudah para pelamar dalam mencari pekerjaan serta mempermudah perekrut dalam mencari pelamar-pelamar. Cara kerja aplikasi ini adalah pelamar dapat menyebarkan CV, Portofolio atau pengalaman kerja mereka dalam aplikasi ini agar perekrut dapat melihat Cv atau portofolio para pelamar serta data diri pelamar sehingga mempermudah perekrut dalam menghubungi pelamar.
